# Visualization of some opencv functions using matplotlib

## Install
```
python -m pip install -r requirements.txt
```

## Visualizations
### Thresholding
```
python vthresh.py -i <image.png>
```

### Morphological operations
```
python vmorph.py -i <image.png>
```
